h1 tag used to add heading of webpage and it can be used once at one webpage

h2 tag used to add sub-heading of the h1 tag, and it can use multiple times in a webpage

h3 tag used to add sub-heading of the h2 tag

h4 tag used to add sub-heading of the h3 tag

h5 tag used to add sub-heading of the h4 tag

paragraph tag used to add paragraph of the sub heading tag in the webpages.
